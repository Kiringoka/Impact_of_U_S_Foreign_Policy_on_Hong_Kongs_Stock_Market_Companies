# Impact_of_U_S_Foreign_Policy_on_Hong_Kongs_Stock_Market_Companies
This repository contains the R code accompanying the bachelor's thesis:
Kolesnikov, D.A. "Impact of U.S. Foreign Policy on Hong Kong's Stock Market Companies (2018–2025)." 
Bachelor's thesis, National Research University Higher School of Economics, Faculty of World Economy and International Affairs, 2026.
The code implements the full empirical analysis presented in the paper, including the difference-in-differences estimation, hypothesis testing, robustness checks, and figure generation.
